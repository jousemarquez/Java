Cree una jerarquía de clases en Java que represente a un arbol familiar,
teniendo que crear una clase para abuelo, otra para padre y otra
para hijo. Puede usar el nombre de clase que estime conveniente, pero tenga
en cuenta que una instancia de la clase abuelo puede representar a un abuelo hombre
o a una abuela mujer, pero solo a uno de ellos. Ocurre lo mismo para la clase que 
representa a un padre y a un hijo.

Todas las instancias de la jerarquía tienen fecha de nacimiento y sexo/género como datos
comunes. También nombre y apellidos.

Además, de forma específica:
- los hijos pueden hacer el vago.
- los padres pueden dar sermones.
- los abuelos pueden contar batallitas.
Nota: implemente esto último como métodos.

De forma adicional:
- todo hijo debe tener dos padres. Debe darse la opción de que sean null si no se dispone de estos datos.
- todo hijo debe tener dos abuelos. Debe darse la opción de que sean null si no se dispone de estos datos.
